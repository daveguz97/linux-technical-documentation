Linux vs Unix

Linux is an Unix clone created by Linus Torvalds. Unix is considered to be the mother of all operating systems. Linux also have a lot of great open source distributions! Ubuntu being the most popular.

These are the 5 most popular distros:

  1. Ubuntu
  2. Elementary OS
  3. Linux Mint
  4. OpenSUSE
  5. Cent OS

Linux has pros and cons compare to other OS like MacOS and Windows.

Pros

  • The best security.
  • Fully Customizable.
  • Open Source.
  • Revive older computer systems.
  • Choose when to allow Software Updates
  • Free to use.
  • Large community support
  • Many distros (some look more like a MacOS, some look more like a windows, and others have their own look.
  • Stability

Cons

  • Certain apps that work with Microsoft or Apple don't work on linux
  • There is a learning curve with linux
  • Office software solution is not as powerful
  • You can't run server-side programs that are specifically designed for windows
  • Can Be difficult to install

With Linux, the fastest way of getting around is to use the terminal. Type CTRL + ALT + T, or use the windows key and type terminal to get started.

Who am I

whoami displays the username in the terminal

username1234@ubuntu:~$
username1234

Sudo

sudo gives you administrative access. sometimes you'll need to type in sudo before you remove something or if your trying to install a software, or even an update.

Make a Directory

To get started, let's create a folder. Folders are known as directory, to create a directory, we type in mkdir folder-name. mkdir stands for Make Directory.

Important! For typing in directories or folders, if you want to include a space, you would have to type in folder\ name

Type mkdir linux-commands in the terminal.

username1234@ubuntu:~$ mkdir linux-commands

Change Directory

To change your directory we use cd. Lets open up the linux-commands directory. And to go back is cd ..

To go to your root directory, you type in cd ~

username1234@ubuntu:~$ cd linux-commands
username1234@ubuntu:~/linux-commands$

Create Files

To create your files we use touch.

username1234@ubuntu:~/linux-commands$ touch example1.txt.

Lets create two more files. and index.html and a style.css file.

username1234@ubuntu:~/linux-commands$ touch index.html style.css

You can create as many files and/or directories as you want by adding a space.

See Your Directories and Files

Sometime we want to list our files and directory.

username1234@ubuntu:~/linux-commands$ ls
example1.txt index.html style.css

You can also see hidden files that start with a . with -a.

ls -a

Open and Code

Lets open the text file we created. In order to do that, we use xdg-open.

xdg-open example1.txt

What if we want to open a file with code editor instead? we use code. Let's code our HTML.

code index.html

Remove

To remove a file you type in rm filename . to rm a directory, you type in rm -r directory.

Important! Don't ever type in sudo rm -rf / because that will delete everything! -f stands for force and it will try to delete everything it could. It's one of the most dangerous commands out there.

Install

You can install and update software with sudo apt-get software

Heres a fun one:

sudo apt-get install cmatrix

Now type in cmatrix in the terminal and see what happens!
